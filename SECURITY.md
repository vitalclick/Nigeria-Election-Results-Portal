# Security policy

See [`docs/SECURITY.md`](docs/SECURITY.md) for the full threat model and
defensive architecture.

## Reporting a vulnerability

Email **security@openballot.ng**. Do not file a public GitHub issue.

We acknowledge within 48 hours and aim to fix critical issues within 7
days. Reporters acting in good faith are credited at
https://openballot.ng/security (post-launch).
